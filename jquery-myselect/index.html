Introduction

JQuery MySelect is yet another multiselect library. This library tries to be simple and easy to integrate in your own project. Everything can be changed by either changing the css (or less) or setting options when calling the library form your javascript code.

Examples

MySelect has different options that can change the appearance and the flow of the plugin. These are some examples of what is possible.

Simple Select box


$("select").myselect({ 
    width: '100%',
    placeholder: 'Countries in Europe'
});

Options

There are a list of options you can use to modify the library to your needs. Here you can find a list with all of them.

Option Default Description
caret_down "▼" (&#9660;) The icon used for the select caret pointing down
caret_up "▲" (&#9650;) The icon used for the select caret pointing down
closeOnClick false Automaticly close the select box after selecting an item. This is only available for single selects.
container_height "34px" The height of the container element. This is the element the user clikcs on to open the dropdown.
delimiter ", " This is the delimiter used to put between selected elements in the container element.
items_max_height "240px" The max height of the items container. When the height of the elements is higher than this, a scroll bar will appear on the right side of the container.
placeholder " - Nothing selected - " This is the default placeholder for all select boxes when nothing is selected. The value can be overridden for every select box by adding a data-placeholder to the select element.
placeholder_x_selected "# selected" The placeholder when more then x_selected_after elements are selected. The # in the string gets replaced by the number of selected elements.
width "240px" The width for both the container and the item container.
x_selected_after 4 The amount of elements that needs to be selected before placeholder_x_selected is displayed.
